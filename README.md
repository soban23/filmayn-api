# filmverse-api
Filmverse is a backend-focused movie platform built using FastAPI and PostgreSQL. The system collects and manages movie data scraped from external sources and exposes it through secure, scalable REST APIs.

The project emphasizes backend architecture, API security, and database optimization, making it suitable for real-world applications.

-Features

Movie data scraping from external websites using BeautifulSoup

Secure JWT-based authentication

User-specific operations (ratings, updates, personalized data)

Clean and well-structured RESTful APIs

Optimized database queries for fast data retrieval
